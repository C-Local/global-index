\# Global Index — Country Explorer



A country search app built with vanilla HTML, CSS (Tailwind), 

and JavaScript, deployed on Netlify with serverless functions 

as a secure API proxy.



\## Features

\- Real-time country search via REST Countries API v5

\- Serverless proxy keeps API key off the client

\- Dynamic membership badges (UN, EU, NATO, G7, G20 etc.)

\- Live embedded OpenStreetMap for each result

\- Neighbouring country grid

\- CRT-inspired UI built with Google Stitch + Tailwind



\## Tech

HTML · CSS · JavaScript · Tailwind CSS · Netlify Functions · 

REST Countries API v5



\## Setup

Requires a REST Countries API key at restcountries.com.

Copy `.env.example` to `.env` and add your key.

Run locally with `netlify dev`.
